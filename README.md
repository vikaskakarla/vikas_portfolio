# Vikas Kakarla - Personal Portfolio

> **Architecting Intellect. Engineering Everything.**

Welcome to the repository of my personal portfolio website! This project is a modern, high-performance, and visually striking web application built to showcase my projects, skills, and achievements as a Master of Computer Science.



## 🚀 Tech Stack

This project is built with a focus on cutting-edge performance, smooth animations, and a sleek developer experience:

- **Framework**: [React](https://reactjs.org/) (via [Vite](https://vitejs.dev/))
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Smooth Scrolling**: [Lenis](https://lenis.studiofreight.com/)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/)
- **Package Manager**: [Bun](https://bun.sh/)

## ✨ Key Features

- **Dark & Neon Theme**: A premium `brand-black` and `brand-neon` color palette that provides a sleek, futuristic vibe.
- **Dynamic Scroll Animations**: Elements fade, slide, and scale into view gracefully using Framer Motion.
- **Sticky Project Cards**: A beautiful stacking card effect for the projects section as you scroll down the page.
- **Magnetic Buttons**: Interactive buttons that playfully pull towards your cursor using custom physics.
- **Fluid Typography**: Responsive typography that scales perfectly across all device sizes.
- **Smooth Scroll**: Native-feeling smooth scrolling via Lenis.

## 🛠️ Getting Started

To run this project locally on your machine, follow these steps:

### Prerequisites

Make sure you have [Bun](https://bun.sh/) installed. Alternatively, you can use `npm` or `yarn`.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/vikaskakarla/vikas_portfolio.git
   cd vikas_portfolio
   ```

2. Install dependencies:
   ```bash
   bun install
   ```

3. Start the development server:
   ```bash
   bun run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.
